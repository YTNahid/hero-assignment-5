Important FAQ

What is the Document Object Model? (DOM)

The DOM (Document Object Model) is a programming interface for web documents. It represents the structure of a webpage as a tree of objects, where each node is an object representing a part of the document, like elements, attributes, or text. This allows programming languages like JavaScript to manipulate the structure, style, and content of the document.

How do you select an element from the DOM?

Elements can be selected from the DOM using JavaScript methods like:

  • document.getElementById('id') – Selects an element by its ID.
  • document.querySelector('.class') – Selects the first element that matches the given CSS selector.
  • document.getElementsByClassName('class') – Selects all elements with the given class name.

What is event delegation in the context of the DOM and why is it useful?

Event delegation is a technique in JavaScript where instead of adding event listeners to specific child elements, you add a single event listener to a parent element. The event is then captured and handled by the parent, which can detect which child triggered the event. This is useful for optimizing performance, especially in dynamic lists where new elements may be added.

How do you manipulate an element's attribute and styles using the DOM?

You can manipulate an element's attributes and styles using DOM methods:

  • element.setAttribute('attribute', 'value') – Sets an attribute to a new value.
  • element.style.property = 'value' – Directly sets a CSS style property on an element.
  • element.classList.add('class-name') – Adds a class to the element's class list.
